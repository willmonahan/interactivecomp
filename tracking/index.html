Tracking Assignment

 For my tracking assignment, I decided to use the AR marker tracking to create a 2D game. I was inspired by the demos we had seen in class, as well as the previous games I had made this semester, so I kind of combined a few ideas to make something new.

 To play my game, you use a marker (the specific marker can be found here) to control the "avatar" (just a circle.) Your goal is to collect as many gold coins as possible, while avoiding the red "bombs." That's it!

 I used a custom marker that I had already created, and used object-oriented programming to create the game itself. The game can be played here.

Enjoy!
